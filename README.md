# Ilarri - Plataforma de Acompañamiento Emocional y Psicología Clínica

![Illari Banner](C:\Users\Antonio\Documents\Residencia\ILARRI\FRONTEND\public\logo1.png)  

## Descripción

**Ilarri** es una plataforma web diseñada para ofrecer **acompañamiento emocional** y servicios de **psicología clínica**.  
Su objetivo principal es proporcionar herramientas accesibles para el bienestar emocional, incluyendo **tests interactivos** que evalúan estados emocionales o niveles de riesgo, almacenando los resultados de forma segura.

## Tecnologías Usadas

### Frontend (React)
- React 19
- React Router v6+
- Framer Motion (animaciones)
- Lucide React (iconos)
- Tailwind CSS
- Axios (peticiones HTTP)
- Otras: react-icons, web-vitals

### Backend
- Python 3.9+
- FastAPI
- SQLAlchemy (ORM)
- Pydantic (validación y schemas)
- Psycopg2-binary (conector PostgreSQL)
- python-dotenv
- Uvicorn (servidor ASGI)

### Base de Datos
- PostgreSQL

## Requisitos Previos

### Frontend
- Node.js ≥ 18.x (recomendado usar **nvm**)
- npm ≥ 8.x

### Backend
- Python ≥ 3.9
- pip
- PostgreSQL instalado y corriendo
- Entorno virtual recomendado (`venv`)
